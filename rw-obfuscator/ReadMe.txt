Notice that in the source code these definitions are default:   
	PARTS_NUM = 6 (A number of parts for assets decomposing)
	VARIANT_KEY = 23 (The key for permutation value)

What is it?
---------------
This is a simple obfuscation program to permutate parts of game assets nested in a IMG archive.

How do use it?
------------------
Just compile it and run the RWObfuscator.exe executable. A syntax is following: RWObfuscator.exe input_file output_file
